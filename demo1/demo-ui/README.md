### Demo Core默认的UI页面

#### 使用方式

1. 引入pom依赖
```xml
<!-- 此功能需要依赖demo-core工程 -->
<dependency>
    <groupId>org.lihao</groupId>
    <artifactId>demo-core</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
<dependency>
    <groupId>org.lihao</groupId>
    <artifactId>demo-ui</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```

2. 服务启动后访问demo-ui.html
  
  
详情请见
  [demo-core](http://192.168.90.8/lihao/demo1/tree/master/demo-core)
